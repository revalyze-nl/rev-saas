# Sample PDF Reports

Place the following PDF files in this folder:

1. `sample-analysis-report.pdf` - A sample AI Analysis report
2. `sample-simulation-report.pdf` - A sample Pricing Simulation report

These files will be available for download in the demo section of the landing page.

## How to Generate Sample PDFs

1. Log in to the dashboard (app.revalyze.co)
2. Run an AI Analysis with sample data
3. Download the PDF and rename it to `sample-analysis-report.pdf`
4. Run a Pricing Simulation with sample data
5. Download the PDF and rename it to `sample-simulation-report.pdf`
6. Place both files in this folder
